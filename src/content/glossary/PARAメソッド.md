# 個人メモ

- 4つのカテゴリで整理
- P: プロジェクト（期限・目標あり）
- A: エリア（継続的な責任範囲）
- R: リソース（参考資料）
- A: アーカイブ（完了・非活性）
- プロジェクトとエリアの混同に注意
- Tiago Forte提唱のメソッド
