# 完了した仕事ノートのアーカイブ方法

## 概要

Obsidianで管理している仕事ノートは、仕事が完了したらメインのタスクリストから移動させ、アーカイブすることで、常にアクティブなタスクリストを見通し良く保つことができる。このアーカイブは、単なる削除ではなく、後から振り返りやすい形で保管することで、過去の経験を未来に活かす資産となる。

## 重要なポイント

-   **アクティブリストの明確化**: 完了したノートを移動させることで、現在取り組むべき仕事や、今後の計画が明確になり、認知負荷を軽減する。
-   **アーカイブの容易さ**: Obsidianの「Move current file to another folder」（現在のファイルを別のフォルダに移動）コマンドを活用することで、キーボードショートカット一つで、完了したノートをアーカイブフォルダへ素早く移動できる。
-   **参照性の確保**: アーカイブされたノートも、時系列に並んだ状態（例：年ごとのフォルダ）で保管されるため、後から過去の経験を参照したい場合に容易にアクセスできる。

## 実践のヒント

-   完了した仕事ノートを移動させる専用のアーカイブフォルダ（例：`Archive/YYYY`）を作成する。
-   「Move current file to another folder」コマンドにショートカットキーを割り当てておく。
-   移動させる際に、完了した日付をファイル名に追記するなど、アーカイブのルールを設けておくと、後からの参照がさらに便利になる。
