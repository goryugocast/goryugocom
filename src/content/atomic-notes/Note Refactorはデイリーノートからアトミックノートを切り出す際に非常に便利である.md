# Note Refactorの使い方

## 概要

Note Refactorは、ノートの一部を選択して別の新しいノートとして切り出すObsidianプラグイン。元のノートには自動的にリンクが残るため、デイリーノートや読書メモからアトミックノートを作る際に非常に便利。

## 基本的な使い方

1. **プラグインをインストール**: Community Pluginsから「Note Refactor」を検索してインストール
2. **切り出したい部分を選択**: テキストエディタで範囲を選択
3. **コマンドを実行**: 「Note Refactor: Extract selection to new note - first line as file name」を実行
4. **新しいノートが作成される**: 選択した部分の1行目がファイル名になり、元の位置にはリンクが残る

## 設定のポイント

- **新しいノートの保存先**: どのフォルダに作成するか設定できる
- **元のページに残す内容**: リンク形式やテンプレートを指定できる
- **ショートカットキー**: Cmd+Shift+N など、使いやすいキーを割り当てる

## 実践のヒント

- 読書メモをアトミック化する際の最重要プラグイン
- 1行目をタイトルにする形式が最も使いやすい
- 切り出した後も、元のノートとのつながりが保たれる
- 複数の部分を連続して切り出すことも可能
