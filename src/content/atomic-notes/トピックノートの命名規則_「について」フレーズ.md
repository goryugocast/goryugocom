# トピックノートの命名規則：「について」フレーズ

トピックノートが増えてくるにつれ、「作成したトピックノート自体を探す」という新たな問題が生じます。この問題を解決する強力なライフハックが、**「〇〇について」という統一フレーズをタイトルに含める命名規則**です。

## 命名規則の基本

```
「ドミナントについて」
「遺伝子について」
「Ableton Pushについて」
「ハーモニー理論について」
```

全てのトピックノートに「について」というフレーズを含めることで、後から「について」で検索するだけで、自分がこれまでまとめたトピックノートの全リストを一瞬で発見できます。

## 効果が顕著になるタイミング

- 100日分のノート: 「について」で検索すれば、100個のノートを見るより早い
- 200日分のノート: 発見性の向上は格段
- 300日以上のノート: 組織化なしではノートを探すのが困難に

## 関連パターン

「について」以外にも活用可能な統一フレーズ：
- 「について調べた」（調査プロセスの記録）
- 「についてまとめた」（完成度の高いまとめ）
- 「について学んだ」（学習過程）

ただし、「について」が最も日本語で他と重複しないため、推奨されます。

## 組織化との関係

ノートの整理が進み、システマティックな構造が完成すると、全文検索の出番は自然と減少します。しかし、このタイミングまでは「について」という検索パターンが強力です。
