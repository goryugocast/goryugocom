# Cmd/Ctrl + Dでデイリーノートを素早く作成する

## 概要
Obsidianのデイリーノート機能を効果的に使うために、ショートカットキーの設定が重要。Cmd/Ctrl + Dはデイリーノートの設定において、最もシンプルで覚えやすいキーバインディングである。

## デフォルト設定
- 何も設定していなければ、Command D（Mac）または Control D（Windows）でデイリーノートが作成される
- 「D」はDaily Noteの「D」として覚えやすい
- このキーが他の機能と競合していない場合は、そのまま使用できる

## カスタマイズ方法

### 設定画面へのアクセス
1. Command + : (コロン)でObsidianの設定画面を開く
2. 左側メニューから「Hotkeys」を探す
3. インクリメンタルサーチで「Daily Note」を検索すると素早く見つかる

### キーバインディングの変更
- 競合していない場合はデフォルトのまま使用
- 競合している場合は別のキーに割り当てる
- Dキーは押しやすい位置にあるため、ショートカット用として相性が良い

## 設定上の注意点
- Command D が何かの機能と競合している可能性がある
- その場合は Command P（パレット）から「Create today's daily note」を実行できる
- 複数の環境で同期する場合は、統一されたキーバインディングを使用すると効率的

## 実装のメリット
- 習慣化しやすい（毎日デイリーノートを開く行為が単純化される）
- ワンキーで今日のノートが開く
- 記録の開始ハードルが低くなる
