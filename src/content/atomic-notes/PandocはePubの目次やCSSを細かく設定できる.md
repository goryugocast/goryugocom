# PandocはePubの目次やCSSを細かく設定できる

## 概要
Pandocは、MarkdownファイルからePub形式の電子書籍を生成する際に用いられる強力なコマンドラインツールである。Web上の簡易な変換サービスとは異なり、コマンドのオプションを指定することで、目次に含める見出しのレベル、文字色や背景色といったCSSによるスタイリングなど、出力結果を細かく制御できる利点がある。

## 重要なポイント
- Pandocを使えば、定型的な変換プロセスをコマンド一つで再現できるため、作業の自動化に適している。
- 専門的な知識が必要だが、ChatGPTのようなAIに質問することで、必要なコマンドや設定方法を学ぶことができる。
- より簡易な方法として、Webサービス「でんでんコンバーター」なども存在するが、カスタマイズ性ではPandocが優れる。
