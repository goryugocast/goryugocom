# デイリーノートをコントロールセンターにする

## 概要

Advanced URIとDataviewを組み合わせることで、デイリーノートが1日の作業を統制する「コントロールセンター」として機能する。やることリスト、ワークスペース切り替えリンク、クイックアクション、メモの受け皿など、全てがデイリーノートに集約される。

## 重要なポイント

**コントロールセンターとしての機能**:
1. **やることリスト** - 上から順番に進めるタスクが並ぶ
2. **ワークスペース切り替え** - 各タスクに対応する作業環境へのリンク
3. **クイックメモ** - Raycastやブックマークレットからの追記先
4. **情報の一時保管** - 後で整理するための受け皿
5. **振り返りの起点** - Dataviewで過去のノートを表示

**1日の流れ**:
1. 朝、デイリーノートを開く
2. 上から順番にタスクをこなす
3. 思いついたことをすぐにメモ
4. 気になるリンクをブックマークレットで保存
5. 夜、1日を振り返る

**デイリーノートの進化**:
- 単なる日記 → 作業の司令塔
- 手動記録 → 自動蓄積
- 静的なページ → 動的なハブ

## 実践のコツ

- テンプレートに全ての機能を組み込む
- 上から順番に進めるシンプルな構造を維持
- 完璧を目指さず、とにかく使い続ける
- 1ヶ月後に効果を実感できる
