# Visual Studio Codeはプログラミング環境の最適解

プログラミングを始めるなら、Visual Studio Code（VSCode）のインストールが第一歩。無料で利用でき、Windows・Mac・Linuxすべてに対応している。ブラウザ版もあるため、iPadやiPhoneでも利用可能。

VSCode自体の情報もウェブ上に非常に多いため、「VSCodeでどうやってやったらいい」という質問をChatGPTにすれば、大抵の操作方法を教えてくれる。

ターミナル機能も内蔵されており、エディタとターミナルを行き来せずに作業できる利便性も高い。まず間違いのない選択肢。
