# デイリーノートから個別ノートへの切り出し手順

## 概要

デイリーノートに雑多にメモした内容から、価値のある部分を個別のアトミックノートとして切り出すプロセス。Note Refactorプラグインを使うと効率的に実行できる。

## 手順

1. **デイリーノートに雑にメモ**: まず思いついたことをデイリーノートに自由に書く
2. **切り出す部分を選択**: 後から見返して、独立したノートにする価値がある部分を選択
3. **Note Refactorで切り出し**: プラグインのコマンドを実行し、新しいノートを作成
4. **リンクを自動生成**: 切り出した部分は自動的にリンクに置き換わる
5. **タイトルを調整**: 必要に応じて、切り出したノートのタイトルを修正

## 実践のヒント

- デイリーノートには完璧な文章を書く必要はない、後で整理すればOK
- 切り出すタイミングは自由（その日のうちでも、後日でも）
- 一度に全部切り出さず、余裕がある時に少しずつでも良い
- タイトルは仮でも構わない、後から何度でも修正できる
