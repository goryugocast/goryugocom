# ファイル移動ショートカットの設定

## 概要

Obsidianの「Move file to another folder」コマンドにショートカットを割り当てることで、ノートを別のフォルダに素早く移動できる。アーカイブ作業や整理作業の効率が大幅に向上する。

## 設定方法

- **設定画面を開く**: Settings > Hotkeys
- **コマンドを検索**: 「Move file」で検索
- **ショートカット割り当て**: 「Move file to another folder」にキーを割り当てる
- **推奨キー**: Cmd+Shift+M など、覚えやすく他と競合しないものを選ぶ

## 実践のヒント

- ファイル移動は頻繁に行う作業なので、ショートカット設定の効果が大きい
- アクティブなノートを直下に置き、落ち着いたらフォルダに移動する運用と相性が良い
- フォルダ候補が表示されるので、入力補完で素早く選択できる
- 複数ファイルの一括移動はできないため、その場合はドラッグ&ドロップを使う
