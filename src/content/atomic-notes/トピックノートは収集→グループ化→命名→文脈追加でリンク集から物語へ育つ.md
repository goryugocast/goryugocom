# トピックノート作成の4ステップ

トピックノートを構築する際の基本的なワークフローです。
1. **収集**：キーワード検索（または「Copy search results」）で、関連する既存ノートのリンクをひとまず集める。
2. **グループ化**：集めたリンクを眺め、似たテーマのものを近くに配置する。
3. **命名**：グループに一言だけ見出し（ラベル）を付け、構造を整理する。
4. **文脈の追加**：各項目に解説や自分の言葉を書き加え、単なるリンク集から「物語」へと育てる。
