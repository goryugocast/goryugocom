# 情報収集における「ゴミを増やさない」思想：ウェブクリッパーとの付き合い方

## 概要
「ウェブクリッパーはゴミを増やすだけ」という過去の経験から、Obsidian Web Clipperを評価する際に「本当に役に立つ情報だけをクリップする」という思想を強調します。ジョニーデシマル的な整理術との組み合わせで、情報の「集める」と「整理する」を両立します。

## 重要なポイント
- 過去の教訓: 従来のウェブクリッパーで「ゴミが増える」という失敗経験から、本当に必要な情報だけをクリップする意識の重要性。
- 情報収集の目的化回避: 単に記事を集めることが目的化しないよう、クリップする情報の質と活用方法を重視。
- ジョニーデシマルとの連携: 体系的な整理術であるジョニーデシマルの思想を取り入れ、情報収集と整理を同時に行う。

## 実践のヒント / 関連する概念
- ウェブクリップする前に、その情報が自分にとって本当に必要か、どのように活用するのかを明確にする。
- Web Clipperのカスタマイズ性を活かし、必要最小限の情報だけを抽出するテンプレートを作成する。
- クリップした情報がジョニーデシマルのカテゴリに収まるように意識し、体系的な整理を心がける。
