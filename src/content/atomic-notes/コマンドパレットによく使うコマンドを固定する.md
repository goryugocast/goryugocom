Obsidianのコマンドパレットでは、よく使うコマンドを上部に固定（ピン留め）できる。キーボードショートカットを設定するほどではないが、頻繁に使うコマンドに便利。

## 固定の方法

Settings → Core plugins → Command palette → Pinned commands で、よく使うコマンドを検索して追加する。

例：「Open today's daily note」を固定

## 固定の効果

コマンドパレット（Cmd+P）を開くと、固定したコマンドが常に最上部に表示される。その下はアルファベット順で表示される。

## 使い分けの基準

- 頻繁に使う + 連続操作が必要 → キーボードショートカット
- そこそこ使うが単発操作 → コマンドパレットに固定
- たまにしか使わない → 固定しない（検索して実行）

## 実用例

デイリーノートを開く操作は毎日使うが、連続して実行することは少ない。このような操作は、キーボードショートカットではなくコマンドパレットに固定すると、覚えるショートカットを減らしつつ、素早くアクセスできる。
