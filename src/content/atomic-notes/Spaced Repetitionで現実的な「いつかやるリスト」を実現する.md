# Spaced Repetitionで現実的な「いつかやるリスト」を実現する

## 概要
GTDの「いつかやるリスト」が破綻するという問題を、Spaced Repetitionプラグインによる自動レビュー間隔調整で解決。暗記アプリと同じ原理で、現実的に処理可能な量のアイテムを毎日提示し、継続可能なシステムを実現する。

## GTDの「いつかやるリスト」の問題点

### 数学的な破綻
```
前提：「いつかやりたいことリスト」は毎週見直す必要がある
- 現在のアイテム数：30個
- 毎週追加：7個（1日1個）
- 1年後のアイテム数：30 + (7 × 52) = 394個
```

- 394個を毎週見直す？ → 実務的に不可能
- 見直しを放棄する → リストの意味が失われる
- 結果的に破綻する

### 心理的な問題
- 膨大な未完了アイテムが目に入る
- 「やらなければ」というプレッシャーが生まれる
- 完了できないアイテムに対する罪悪感

### David Allen（GTD著者）の告白
- 本人も「いつかやるリストだけは破綻する」と認めている
- ただし「あるとないとではモチベーションが違う」
- つまり、破綻することは織り込み済み

## Spaced Repetitionでの解決

### 基本原理
- 暗記アプリ（Anki など）と同じ仕組み
- ユーザーの「できた/できない」判定に基づき、復習間隔を調整
- 忘れかけた時期に自動的に復習対象に上がる

### レビュー間隔の例
```
初回提示：　　　　　「このアイテムを今日やりますか？」
「また今度」→ 3日後に再提示
「また今度」→ 10日後に再提示
「また今度」→ 30日後に再提示
「完了」　　→　学習対象から外れる
```

## GTDの破綻から現実的なシステムへ

### 従来のGTD：「いつかやる」の悲劇
```
「いつかやる」リスト（300個）
↓ 週単位で見直し要求
「えーと、これ、やる？」× 300個
↓ 実務的に不可能
破綻
```

### Spaced Repetition活用：現実的なアプローチ
```
「いつかやる」リスト（300個）
↓ Spaced Repetition で自動管理
「今日はこの 3～5 個だけ見直して」
↓「できた」または「また今度」を選択
↓ 自動的に次回提示日を計算
↓ 継続可能
```

## 実際の運用フロー

### 毎日の流れ
1. Spaced Repetition プラグインを起動
2. 「今日のレビュー対象」が 3～10 個程度表示される
3. 各アイテムについて判定：
   - 「やります」→ その場で実行
   - 「また今度」→ 自動的に数日後に再表示
   - 「完了」→ リストから外れる

### 忙しい日の対応
- 提示されたアイテムが全て「また今度」でいい
- レビュー対象が次の日に延期される
- 本来の仕事を優先できる

### 時間に余裕がある日
- 3～5 個をレビュー
- 実際に「やります」を選択して実行
- 段階的に「いつかやる」が減っていく

## メリット

### 心理的な軽減
- 300個を一度に見ない
- 毎日 3～5 個だけに集中
- 現実的に処理できる感覚

### 自動優先順位付け
- 何度も「また今度」が繰り返されるアイテムは自動的に重要度が下がる
- 本当にやりたいアイテムは「やります」を選択
- 自然と優先順位が決まる

### 実績の可視化
- 「今週は何を完了したか」が見える
- 達成感が得られる
- モチベーション維持につながる

## 実装時の注意点

### #reviewタグの重要性
- 「#review」タグが付いたノートだけをSpaced Repetitionの対象にする
- 全てのノートをレビュー対象にするのは非現実的
- セレクティブに絞ることが継続のコツ

### パラメーターの調整
- デフォルト設定でも機能するが、自分の頻度に合わせて調整可能
- 「毎日 10 個見たい」なら、逆算してパラメーターを設定

### 手動レビューとの併用
- Spaced Repetition はツール
- 時々「全体を眺める」手動レビューも有効
- 1ヶ月ごと、四半期ごとの見直しも検討

## GTDとの違い

### 従来のGTD
- 理想的だが、実践は難しい
- 特に「いつかやるリスト」は現実的に破綻しやすい
- 完璧さを目指すとストレスになる

### SpacedRepetition統合版
- 現実的に処理可能な量だけを毎日提示
- 「できない日」があってもシステムが対応
- 現実のペースに合わせたフレキシブルな管理

## 注意点

### 「完了」の定義
- 「完了」を選択すると、学習対象から完全に外れる
- 本当に不要になるまで「また今度」で先送りしてもいい
- 無理に「完了」を選択する必要はない

### 見直しの頻度
- 毎日 Spaced Repetition を起動する習慣をつける
- あるいは、週 2～3 回でも効果がある
- ただし「継続」することが最重要
