Obsidian Syncを利用している場合、iCloudでの設定分離とは異なるアプローチで、デバイスごとの設定を管理できる。Syncの設定画面にある「Vault Configuration Sync」セクションで、どの設定項目を同期するかを個別に制御できる。

主な設定項目には以下がある：
- Main settings（メイン設定）
- Appearance（外観）
- Core plugin settings（コアプラグイン設定）
- Community plugin settings（コミュニティプラグイン設定）
- Active plugins list（有効なプラグインのリスト）

例えば、PCでは「Properties view」プラグインをオフにしてハードコア派の設定にし、モバイルでは同じプラグインをオンにしてイージーモードにする、といった使い分けが可能になる。これにより、各デバイスの特性に合わせた最適な環境を維持しつつ、ノート本体は同期できる。
