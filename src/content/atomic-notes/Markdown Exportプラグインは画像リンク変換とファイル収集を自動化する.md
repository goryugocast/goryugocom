# Markdown Exportプラグインは画像リンク変換とファイル収集を自動化する

## 概要
Obsidianのコミュニティプラグインである`Markdown Export`は、Obsidian独自のWikiリンク形式(`![[image.jpg]]`)で埋め込まれた画像を、外部ツールでも解釈可能な標準のMarkdown形式に変換する機能を持つ。同時に、ノート内で使用されている画像ファイルを指定した一つのフォルダに自動で収集してくれるため、電子書籍の入稿データ作成など、外部にファイルを書き出す際の煩雑な作業を大幅に効率化する。

## 重要なポイント
- このプラグインにより、画像が多い原稿でも、手作業でリンクを修正したり画像を探し回ったりする必要がなくなる。
- 執筆中はObsidianの便利なWikiリンクを使い続け、最終的な書き出しの段階でのみ変換を実行できる。
- プラグインの設定で、書き出し先のフォルダをあらかじめ指定しておくことができる。
