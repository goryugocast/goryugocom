「Publish current file」はObsidian Publishで開いているノートのみを素早く公開できるコマンドである。

## スマートフォンでの活用

- コマンドパレットから「Publish current file」を実行
- 開いている1つのノートだけが公開対象になる
- 複数ファイルを指定する場合と違い、迷わない

## 効率化のコツ

- よく使うコマンドなので、コマンドパレットでピン留めして一番上に固定
- スマホから「下にスワイプ → コマンドパレット開く → ピン留めされたPublishコマンドをタップ」で素早く完了

## 実務的なワークフロー

- デイリーノートや記事を更新したらすぐPublish
- モバイルからの更新も容易に実行可能
- 複数の公開対象を管理する手間が最小限
