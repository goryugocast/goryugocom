# Copy search results：関連ノート一覧を自動生成

Obsidianの最強の機能の一つ「Copy search results」は、検索結果に表示されたノート群へのリンクを一瞬でクリップボードにコピーできます。この機能がなければ手作業で一つ一つリンクを集める必要があり、トピックノート作成の効率は大きく異なります。

## 使い方の基本

1. キーワード（例：「遺伝子」）で検索
2. 結果に表示される件数表示部分をクリック
3. 「Copy search results」メニューを選択
4. クリップボードに検索結果のリンク一覧がコピーされる

## 出力形式のオプション

- **Markdown Link**: `[タイトル](note-link)` 形式
- **Wiki Link**: `[[タイトル]]` 形式
- **Text only**: タイトルテキストのみ

Markdownリンク形式を使うと、新規ノートに貼り付けた際に外観が統一されます。

## 実践的な活用シーン

### トピックノート作成の流れ

1. **検索実行**: 「ドミナント」で検索 → 193個の結果
2. **リンク抽出**: Copy search resultsで一覧を自動生成
3. **ポップアッププレビュー**: Cmd/Ctrlキーで内容を確認
4. **厳選と統合**: 不要なリンクを削除し、内容を整理

この一連のプロセスこそが、Obsidianを使った知的生産の核となります。

## 手作業の経験が価値を高める

初心者は「一度は手作業でリンクを一つ一つ集める苦労を経験する」ことで、この機能の本当のありがたみを理解できます。最初の不便さがあるからこそ、効率化の価値を実感できるのです。
