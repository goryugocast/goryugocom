# ウェブクリッパーに求める機能：自由なクリップとテンプレート化

## 概要
ウェブクリッパーに求められる機能として「ウェブページの自分の好きな部分だけを保存する」ことと「それをテンプレート化できる」ことを強調します。Obsidian Web Clipperが、このニーズを超簡単に実現できるツールであることを、Amazonやサウンドハウスのクリップ例を挙げて説明します。

## 重要なポイント
- 自由なクリップ: ページ全体ではなく、ユーザーが必要とする情報のみをピンポイントで抽出・保存できること。
- テンプレート化: クリップした情報を、あらかじめ設定した形式（マークダウン）で自動的に整形できること。
- 効率的な情報保存: Amazonの書誌情報やサウンドハウスの楽器情報など、特定のサイトから定型的な情報を収集する際に特に有効。

## 実践のヒント / 関連する概念
- 頻繁に情報を取得するウェブサイトごとに、専用のWeb Clipperテンプレートを作成する。
- テンプレート内でCSS Selectorなどを活用し、必要な情報だけを抽出するルールを設定する。
- Web Clipperで保存した情報が、Obsidian内でどのように活用されるかを想定してテンプレートを設計する。
