## 概要

Base プラグインは複数のビュー（テーブル）を一つのBase内に持たせることで、**異なる条件での表示切り替えをタブ形式で実現**できる。

## 実装パターン

### 例：年度別表示

```base
# 2025年のノート一覧
フィルター: filename starts with "2025"
```

別のテーブルとして：

```base
# 2024年のノート一覧
フィルター: filename starts with "2024"
```

## ユースケース

1. **年度管理**: 2024年/2025年の切り替え
2. **ステータス管理**: 進行中/完了/未開始の切り替え
3. **優先度管理**: 高/中/低の優先度別表示
4. **担当者別**: 特定のpeople値ごとの記録

## ビュー間の相互参照

- 同一Baseで複数フィルター条件を持つ
- タブ形式で切り替え可能
- 一度の設定で複数条件を管理

## DataViewとの比較

Notionなど他ツールのデータベース同様の機能を、Obsidian内で実現。

- **Notion**: テーブル/ギャラリー/カレンダーなど複数ビュー
- **Base**: テーブル型で複数ビューを内包

## 並べ替えの活用

- ファイル名の先頭に日付を付与
- 並べ替え：A→Z または日時順
- 自動的に時系列並べ替えが可能
