# Publish current fileコマンドで単一ファイルを高速公開する

「Publish current file」コマンドを使うことで、開いているノートのみをすぐに公開できる。

## 利用方法

**コマンドパレット経由でアクティブなファイルを公開**

1. 公開したいノートを開く
2. コマンドパレットを起動（Cmd/Ctrl + P）
3. 「Publish current file」を実行
4. その他のファイルには影響しない

## モバイル対応での活用

スマートフォンから効率的にサイト更新ができる。

- 外出先からも素早く公開可能
- 個別ファイルの公開だけで完結
- 不要な設定画面へのアクセスが不要

## スマートフォンでの実践的な流れ

1. 下にスワイプでコマンドパレットを開く
2. ピン留めされたPublishコマンドをタップ
3. 更新完了
