# Windowsでもランチャーツールで同じことができる

## 概要

RaycastはMac専用だが、Windowsにも同様のランチャーツール（PowerToys Run、Woxなど）があり、Advanced URIと組み合わせてデイリーノートへの素早い追記機能を実現できる。OSに関わらず、同じワークフローを構築できる。

## 具体的な活用方法

**Windows向けランチャーツール**:
- PowerToys Run（Microsoft公式）
- Wox（オープンソース）
- Keypirinha
- Alfred for Windows（非公式）

**実装方法**:
1. ランチャーツールをインストール
2. カスタムコマンドやプラグインを設定
3. テキスト入力 → URLエンコード → Advanced URI起動
4. デイリーノートに自動追記

**Mac版との共通点**:
- 基本的な仕組みは同じ
- JavaScriptやPythonスクリプトを活用
- Advanced URIの呼び出し方は変わらない

## 実践のコツ

- Macの例を参考にWindowsで再現できる
- ランチャーツールのドキュメントを確認
- コミュニティで同じことをしている人の例を探す
- ChatGPTに「Windows版のRaycastのような実装方法」を聞く
