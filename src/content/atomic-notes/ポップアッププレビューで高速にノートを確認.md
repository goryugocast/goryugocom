# ポップアッププレビューで高速にノートを確認

Cmd/Ctrlキーを押しながらノートリンクにカーソルを合わせると、ノートの内容がポップアップ表示されます。これにより、リンク一覧を見ながら、わざわざノートを開かなくても内容を確認できる強力なテクニックです。

## 活用シーン

### Copy search resultsと組み合わせた使い方

1. 検索で関連ノート193個を発見
2. Copy search resultsで一覧をコピー
3. 新規ノートに貼り付け
4. **Cmd/Ctrlキーを押しながら各リンクにカーソルを合わせ、内容をプレビュー**
5. 不要なリンクを削除、重要な情報を整理

### トピックノート作成時の意思決定を加速

どのノートに重要な情報が含まれているかを、**わざわざノートを開く手間なく判断**できます。このステップの時間短縮が、全体の作業効率に大きく影響します。

## 手動で開くフロー との違い

### 従来の方法（非効率）
リンククリック → ノート開く → 内容確認 → 戻る → 次のリンク（繰り返し）

### ポップアッププレビュー（効率的）
Cmd/Ctrl + カーソル移動 で連続確認 → 決定 → 削除・編集（1度だけ）

## 他の機能との連携

- デイリーノートでの日付確認
- 複数ファイルにまたがるコンテンツ確認
- ブログの参考記事検索

Obsidianの標準機能として備わっているこの仕組みは、プラグイン不要で動作します。
