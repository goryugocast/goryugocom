線を使って「タイトルと本文を分ける」、色を使って「自分の内からの情報と外からの情報を分ける」、数字を使って「順序や優先順位を分ける」、3種類の分けて書く方法です。

今回は、「**分けたパーツをつなぐ技術**」を紹介します。

## 🧩 分けたものをつなぐ重要性

前回は、アイデアや情報を整理するために「分ける」手法について紹介しました。線、色、数字といった方法を使うことで、複雑な内容を分けて考えることができるようになります。

[

![](https://substackcdn.com/image/fetch/w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F0ac7b088-a450-488a-ad98-88a25cb25e06_2266x1188.jpeg)



](https://substackcdn.com/image/fetch/f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F0ac7b088-a450-488a-ad98-88a25cb25e06_2266x1188.jpeg)

「分けて」書くことで、頭の中のごちゃごちゃした思考も整理され、何が大事なのかが自然と浮かび上がってくるからです。

しかし、分ける「だけ」では十分とは言えません。次の段階では、分けた要素がどのようにつながり、全体としてどのように機能するのかを理解することが重要になってきます。

分けることで、情報やアイデアがパーツ化され、そのパーツを「つなぐ」ことで、それぞれの関係性が視覚化され、さらに深く考えることができるようになるのです。

## 🔗 3つの「つなぐ」技術

情報をつなぐ際の基本となるのが、次の3つの記号です。

### →（矢印）で「順序」や「因果関係」を示す

矢印は、情報の「順序」や「因果関係」を示すのに適した記号です。

たとえば、順序を表す時はこのように表現できます。

- 朝のルーチン：起床 → 歯磨き → 朝食 → 出発
    
- プロジェクトの進行：アイデア出し → 企画 → 実行 → 振り返り
    

[

![](https://substackcdn.com/image/fetch/w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Ff6b6d4c1-ec95-4e1d-a164-53a4a952562e_2266x1488.jpeg)



](https://substackcdn.com/image/fetch/f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Ff6b6d4c1-ec95-4e1d-a164-53a4a952562e_2266x1488.jpeg)

矢印を使うと、連続的な流れやストーリー性が可視化されるため、物事の前後関係が直感的に理解しやすくなります。

他にも、ある原因が結果を生み出す流れを矢印でつなぐこともできます。接続詞の「つまり」「だから」「なぜなら」「その結果」といった言葉を矢印で置き換えるイメージです。

原因と結果を矢印で表すと、このようになります。

- 風邪をひいた流れ：薄着で外出 → 体が冷える → 風邪をひく
    
- 環境への影響：工場の排気ガス増加 → 大気汚染による健康被害 → 医療費の増加
    

[

![](https://substackcdn.com/image/fetch/w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Fa5546202-4a6e-4d78-a3a7-aa7f54eb3103_2266x1488.jpeg)



](https://substackcdn.com/image/fetch/f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Fa5546202-4a6e-4d78-a3a7-aa7f54eb3103_2266x1488.jpeg)

原因と結果が連続的に影響し合う流れを視覚的に矢印で表現することができます。これにより、複雑な因果関係が一目で理解しやすくなります。

### ＝（イコール）で「同じ性質」や「関連性」を示す

イコール記号は、要素間の「同じ性質」や「関連性」を示すのに役立ちます。  
この記号を使うことで、異なる情報が本質的に同じ意味を持つことを強調できます。

- 「安い商品」＝「コストパフォーマンスが良い」
    
- 「決断を下す判断基準」＝「羅針盤」
    
- 「生活を豊かにする趣味」＝「人生のスパイス」
    

[

![](https://substackcdn.com/image/fetch/w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F8e54ad6f-0ab0-4089-b48c-dc7262e41c4c_2266x1488.jpeg)



](https://substackcdn.com/image/fetch/f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2F8e54ad6f-0ab0-4089-b48c-dc7262e41c4c_2266x1488.jpeg)

要素間の「同じ性質」や「関連性」などの等価性を示すことで、異なる概念がつながり、全体像の理解が進みます。

### ―（補助線）で「補足情報」を加える

補助線は、要素の補足を示すのに便利です。これはメモの中で小さな注釈を付けるような感覚で使えます。

- 「健康」―運動、食事、睡眠
    
- 「アイデア」―ブレストで発生した発想の一部
    

[

![](https://substackcdn.com/image/fetch/w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Fae6cc2ad-2038-4ab7-8240-0d523da4ba4c_2266x1488.jpeg)



](https://substackcdn.com/image/fetch/f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Fae6cc2ad-2038-4ab7-8240-0d523da4ba4c_2266x1488.jpeg)

補助線を使うことで、要素に説明を追加することができます。また、補助線は書き込むスペースが限られているときに、少し離れた空いたスペースにメモを書くためにも役立ちます。

これらの記号を組み合わせることで、分けた要素がどのように関係しているのかをわかりやすく表現できます。「つなぐ」技術が身につくと、全体の構造が明確化でき、情報の整理が格段にスムーズになります。

## ✏️ 書いて考える力を強化する

どんな情報も、完全に独立して存在することはほとんどなく、何かしらの関係性を持っています。そして、その関係性は、コツを掴めばシンプルな記号で表現することができます。

[

![](https://substackcdn.com/image/fetch/w_1456,c_limit,f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Facc78249-75be-4384-9361-df03e12f2210_1920x1080.jpeg)



](https://substackcdn.com/image/fetch/f_auto,q_auto:good,fl_progressive:steep/https%3A%2F%2Fsubstack-post-media.s3.amazonaws.com%2Fpublic%2Fimages%2Facc78249-75be-4384-9361-df03e12f2210_1920x1080.jpeg)

書くことで、考えをまとめたり、考えを深めることが簡単になる理由は、シンプルな記号を使って**関係性を表すことで、物事を抽象化して捉え直すことができる**からです。

やっていること自体はそんなに複雑ではないのですが、頭の中だけで処理しようとすると途端に難しくなります。だからこそ、書き出すことが重要なのです。書き出すことで、頭の中は「考えること」に集中できるようになります。

また、要素と要素をつなげようと意識するだけで、今まで思いつかなかったことを思いついたり、自分の言葉で考えをまとめることができるようになります。

「分ける」技術と「つなぐ」技術を組み合わせることで、情報整理はよりスムーズになり、思考全体を俯瞰する力＝「考える力」が育ちます。

ということで今日は「**情報同士をつなぐ技術**」というお話でした。
